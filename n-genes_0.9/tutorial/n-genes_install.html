1. Introduction

N-genes is a complete genetic algorithm/programing framework written in Java. Its main advantages are:

  • Simple and componentized API, allowing a very fast implementation of various problems, operators, selectors, etc. ;

  • Light and human readable XML config files, which can instantiate every user written components, through dynamic class loading'and 'introspection technologies;

  • Various optimisation techniques like objects pooling and multi-level caches.

2. Requirements

2.1. Software

N-genes is a framework built on the Java J2SE 5.0 plateform. To check if everything is installed you can type on a shell:

$ java -version
java version "1.5.0_02"
Java(TM) 2 Runtime Environment, Standard Edition (build 1.5.0_02-b09)
Java HotSpot(TM) Client VM (build 1.5.0_02-b09, mixed mode)

The java version returned must be greater or equals to "1.5.0". You can download an up-to-date JDK on Sun site.

2.2. Hardware

The whole framework takes 14 MB of hardisk once installed. But if you delete the source and documentation directories (src/ and doc/), the needed disk space will fall under 2MB.

There are no particular requirements on RAM but the most you have, better the performances will be. We advise at least 256 MB, but 1GB is better.

3. Installation

3.1. Installation on Linux and other Un*x

  • Download the latest tarball at http://cui.unige.ch/spc/.

  • Copy it where you want to install your apps.

  • Unarchive with: 

$ tar -jxvf n-genes_<version>.tar.bz2

3.2. Installation on MS-Windows

  • Download the latest tarball at http://cui.unige.ch/spc/.

  • Copy it where you want to install your apps.

  • Use your favorite program to expand it. If you hesitate, you should try 7-zip (open source).

4. What's Included ?

n-genes is organised as follows:

n-genes.jar

the n-genes framework runtime;

build.xml

an ant building file for the whole project;

config/

an directory containing the DTD and xml-schemas needed by the xml config files amd differents exemples.

doc/

the API JavaDoc directory — to read it, point your web browser to <installPath>/doc/index.html;

src/

the sources directory;

tutorial

some tutorials introducing n-genes (same as those on the web site).

5. Usage

First, you need to add the n-genes jar to your CLASSPATH variable. Then you can invoke n-genes with:

$ java [vm_flags] ch.unige.ng.bin.NGenes --file <config_file.xml>

where <config_file.xml> is the name of your config file (required) and [vm_flags] the optional java virtual machine flags. We advise to use -server wich provides deeper hotspot optimisations and -Xmx / -Xms to augment the maximum/initial heap size.

If you choose to implement you own classes, we advice to begin a new package tree and add it to the CLASSPATH variable.
